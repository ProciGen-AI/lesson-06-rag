# Fundamental Resources

## Manager Starting Points

Every NovaOps manager should know where the basic operating resources live. The employee handbook explains company-wide policies. This playbook explains manager practice. BambooHR holds employee records, reviews, leave requests, and HR workflows. Jira, Salesforce, GitHub, and AWS contain operational work depending on the department. Slack and Webex support day-to-day communication.

Use [the NovaOps wiki](https://wiki.novaops.example) for team charters, decision records, onboarding plans, and department operating procedures. Use [BambooHR](https://bamboohr.novaops.example) for HR records. Use [the IT request portal](https://it.novaops.example) for access, device, and software requests. Use [billing review](https://billing.novaops.example) when a request may create new paid spend.

## Escalation Examples

If a report asks for GitHub access and they are a contractor, route to IT with manager approval, repository scope, and end date. Do not add them to a broad team because they are blocked.

If a Customer Success employee cannot join a customer call because Webex seats are constrained, open an IT ticket and tell the manager of the business impact. If expansion creates spend above threshold, Finance must review.

If an employee says they need medical leave, direct them to People Operations and BambooHR. Discuss coverage and timing, but do not ask for diagnosis details.

If a manager wants to buy a new SaaS tool, document the business need, data involved, expected users, monthly or annual cost, and whether Okta SSO is available. Then involve IT and Finance before company data enters the tool.

If performance concerns become serious, contact People Operations before saying the employee is on a formal warning or that their job is in jeopardy.

## Manager Records

Managers should keep work records in work systems and employment records in HR systems. Jira, GitHub, Salesforce, and project documents are evidence of work. BambooHR and People Operations workflows are for formal reviews, correction plans, leave, and sensitive employment decisions. A private notebook can help a manager remember facts, but it should not become the only record of feedback or decisions.

When in doubt, ask where a record belongs before writing it. Sensitive records are easier to protect when they start in the right place.
